해당 오류는 윈도우 내 IME 호환성 문제때문에 발생할 수 있어요.
아래 해결방법을 참고하여 문제를 해결해 보시길 바랍니다.

로그인 시 아이디가 이미 사용중이라고 나와요.
이전 접속 상태에서 정상 종료가 되지 않았다면, 이후 접속 시도 시 해당 문구가 발생될 수 있습니다.
[해결방법]
1. 인터넷 창 실행 → 도구 → 인터넷 옵션 → 검색기록 → 삭제 → 임시인터넷 파일, 쿠키 선택 삭제
2. 인터넷 옵션 → 고급 선택 → '프록시 연결 ~'의 체크 해제 → 확인 (이용하시는 인터넷 브라우저에 따라서 쿠키 삭제 경로가 다르니, 인터넷으로 이용 브라우저를 검색하여 참고 부탁드릴게요!)
